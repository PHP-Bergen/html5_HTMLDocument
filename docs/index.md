# The new HTML parser

## Dom\HTMLDocument &  Dom\XMLDocument

The new HTML Parser introduced in php8.4 makes life for anyone who needs to parse HTML with php.

Working with the web, we often need a safe and standardised way of working with:

* URL and URI 
* The ability to parse/create HTML Documents.

### Dom\HTMLDocument & Dom\XMLDocument were introduced in PHP 8.4 to address theses issues.


* [PHP RFC: DOM HTML5 parsing and serialization](https://wiki.php.net/rfc/domdocument_html5_parser) 
* [PHP RFC: Focussing on CSS selector support](https://wiki.php.net/rfc/dom_additions_84)

### The URI parsins api was introduced in php8.5

* [introduced in PHP 8.5](https://wiki.php.net/rfc/url_parsing_api)
